# Student Report Card Application

A C# console application built to manage student marks, calculate totals, and display ranked report cards.


## Features
- Input the number of students
- Store each student's name and marks for 3 subjects (English, Math, Computer)
- Automatically calculate total marks
- Rank students in descending order of total marks
- Display detailed report cards with positions


## How to Run
1. Open the terminal in your Codespace
2. Compile the code with:  
   csc Task5.cs
3. Run the executable with:  
   ./Task5.exe (or just Task5 on some systems)


## Project Structure
- Task5.cs: Main C# code file with all functionality
- README.md: Project documentation
